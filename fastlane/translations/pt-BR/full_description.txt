Adiciona um botão ao painel de ajustes rápidos para fazer as Capturas da tela.

Após a instalação, você precisa adicionar um botão de ação ao menu de ajustes rápidos e conceder as permissões para poder fazer as Capturas da tela e salvar as imagens no armazenamento do dispositivo.

Se o app apresentar problemas no Android 9/Pie ou 10, você pode tentar "Usar o método integrado de captura da tela" em configurações do app.

<h2>Funcionalidades:</h2> 
✓ Capturas da tela diretamente dos ajustes rápidos
✓ Sem privilégios root
✓ Notificação após cada Captura da tela (pode ser desativada)
✓ Pode imediatamente compartilhar, editar ou excluir a Captura da tela via notificação
✓ Botão flutuante, tipo um balão do chat (Android 9+)
✓ Use como app de assistência para fazer uma captura de tela (toque longo no botão de início)
✓ Capturas da tela de uma área específica (toque longo no botão)
✓ Pode definir um adiamento para cada Captura da tela
✓ Salve em qualquer pasta ou qualquer tipo de armazenamento, p.ex. cartão SD
✓ Salve em diferentes formatos de arquivo: png, jpg ou webp
✓ Faça as Capturas automáticas via apps como Tasker ou MacroDroid
✓ Gratuito, de software livre e sem anúncios

Esta é uma modificação do "Screenshot Tile [Root]" mas não exige privilégios root. Se você tiver acesso root, deveria usar o <a href="https://play.google.com/store/apps/details?id=com.github.ipcjs.screenshottile">app original</a> do ipcjs, é mais rápido e mais estável que esta modificação.

<h2>Permissões:</h2>❏ <i>android.permission.WRITE_EXTERNAL_STORAGE "Fotos/Mídia/Arquivos e Armazenamento"</i>
Isto é necessário para salvar os arquivos de Captura da tela no armazenamento interno do seu dispositivo.

❏ <i>android.permission.FOREGROUND_SERVICE</i>
Desde o Android 9/Pie, esta permissão é necessária para fazer as Capturas. Isso basicamente significa que este aplicativo pode estar ativo em segundo plano. Entretanto, o aplicativo sempre mostrará uma notificação quando estiver sendo executado.
