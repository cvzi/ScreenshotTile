在快速设置面板中添加一个按钮/图块以截取屏幕截图。

安装后，您需要将按钮/图块添加到快速设置，然后授予录制屏幕并保存图像到内部存储的权限。

<h2>特点：</h2>✓ 从快速设置中获取屏幕截图
✓ 不需要 root 权限
✓ 屏幕截图后通知（可禁用）
✓ 从通知中立即分享、编辑或删除屏幕截图
✓ 使用包含的图像编辑器编辑截图
✓ 浮动按钮/叠加按钮，类似聊天气泡 (Android 9+)
✓ 用作助理应用以进行截图（长按主屏幕按钮）
✓ 只对屏幕上的某个特定区域进行屏幕截图（长按磁贴）
✓ 延迟屏幕截图
✓ 存储在任何存储中的任何文件夹，例如： SD 卡
✓ 以不同的文件格式存储：png、jpg 或 webp
✓ 使用 Tasker 或 MacroDroid 等应用自动截图
✓ 免费，开源，没有广告

这是一个 "Screenshot Tile [Root]" 的分叉但不需要 root 权限。 原应用： <a href="https://github.com/ipcjs/ScreenshotTile">github.com/ipcjs/ScreenshotTile</a>

<h2>权限：</h2> <i>android.permission.WRITE_EXTERNAL_STORAGE "Photos/Media/Files and Storage"</i>
这是在您设备的内部存储中保存屏幕截图文件所必需的。

❏ <i>android.permission.FOREGROUND_SERVICE</i>
从 Android 9/Pie 起需要此权限进行屏幕截图。 这基本上意味着这个应用可以在不显示自己的情况下运行。 然而，应用将在运行时始终显示通知。 这基本上意味着这个应用可以在不显示自己的情况下运行。 然而，应用将在运行时始终显示通知。
